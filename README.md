The platform is designed to centralize the discovery, organization and participation in real-life community events such as:
- trekking and outdoor activities
- sports activities
- dinners and social meetups
- cultural experiences

The goal is to replace the current fragmented communication channels (WhatsApp, Instagram, manual coordination) with a single structured platform that manages:
- event discovery
- user registration
- participant management
- payments
- organizer tools
- community engagement

The platform must be:
• mobile-first
• simple and intuitive
• scalable for growing communities
• usable directly from a smartphone
• installable as a Progressive Web App (PWA)


This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
